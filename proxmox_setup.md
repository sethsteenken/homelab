# Proxmox Setup

## Download

Download [ISO of Proxmox Virtual Environment](https://www.proxmox.com/en/downloads/proxmox-virtual-environment/iso)

Flash the ISO to a thumb drive using [Rufus](https://rufus.ie/en/).

Boot from thumb drive and follow install instructions.
